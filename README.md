# Teacher Scheduling System

A simple PHP & MySQL teacher scheduling system using XAMPP.

## Features
- Teacher registration & login
- Admin panel
- Schedule management
- Role-based access

## Setup Instructions
1. Clone the repository
2. Copy project to `xampp/htdocs`
3. Start Apache & MySQL
4. Import database from `database/teacher_schedule_db.sql`
5. Visit `http://localhost/teacher-scheduling`
